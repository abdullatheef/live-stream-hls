Code Block Highlighting with Copy Button

This is an example of client-side syntax highlighting using Prism.js with a copy button for each code block.

JavaScript Example

      
function greet(name) {
  return `Hello, ${'${name}'}`;
}

console.log(greet("World"));

Python Example

      
def greet(name):
    return f"Hello, {name}!"

print(greet("World"))
